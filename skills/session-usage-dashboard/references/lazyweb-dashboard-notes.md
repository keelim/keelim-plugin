# Lazyweb Dashboard Notes

Use these notes when styling the generated offline dashboard.

- Keep the first viewport operational: title, scope, generated time, and four KPI cards.
- Use ranked usage tables for tools, skills, and subagents instead of decorative charts that require libraries.
- Use compact horizontal bars inside tables for quick scanning.
- Include a concise activity timeline with sanitized event names only.
- Prefer restrained SaaS analytics patterns: clear sections, strong hierarchy, muted borders, and no marketing hero.
- Do not embed Lazyweb URLs, signed screenshot URLs, image assets, fonts, or external scripts in the generated file.
